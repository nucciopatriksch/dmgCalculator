The calculator is intuitive to use, the main value that is mandatory is the "Class Skill Damage" field.
Then simply compile the other fields, only the ones you need. This calculator is capable to give two kinds of output, one specifically for those skill that deal damage once every tot seconds, the last are time based and for this you will enable the menu in the section Time based Damage to calculate continuous damage.

Il calcolatore è intuitivo da usare, il valore principale obbligatorio è il campo "Class Skill Damage".
Quindi compila semplicemente gli altri campi, solo quelli che ti servono. Questa calcolatrice è in grado di fornire due tipi di output, uno specifico per quelle abilità che infliggono danno una volta ogni tot secondi, gli ultimi sono basati sul tempo e per questo abiliterai il menu nella sezione Time based Damage per il calcolo del danno continuo.

Rules for using the calculator

Regole per l'utilizzo del calcolatore

Compile the "Class Skill Damage" field, it refers to the damage caused by one of the skills of a class; check the actual damage value from the game itself via the Abilities tab before entering.
Then insert the reference values of your character's stats; you can also use values that are not in your possession to make a comparison between one or more configurations.
The calculation of the damage can be done without the need to necessarily enter all the values; just think that some of the multipliers such as "Damage when combat starts" are momentary bonuses.
To calculate critical damage of a single skill, the "Critical damage bonus" field must be compiled, having a value greater than 0.
If "Time based Damage" menu is enabled, Critical hits will be calculated through a random chance based on "Critical Chance" field that will appear.

As you can see the second menu at center contains other fields referring to other bonuses such as Weapon's stats, Atlas buffs, Verdict and so on.


Note:
- All the stats are hard coded; using bigger values that are unintented will have no effect to the final calculation.
- "Elder Multiplier" increases the damage done by a certain skill of a class, obtained by progressing within the Halls of Elder Gods (max 28%).
- "Generic Class Buff" field refers to a class skill that increases damage, like paladin's Holy Ground skill.
- "Aelerium-9 Talent" field refers to a class talent that can increase damage.
- "Symbol Bonus" field refers to a class symbol that can increase damage.
- "God Form" checkbox simulate the character's divine form, resulting in an increased damage output. Other fields connected to the divine state will appear.
- Atlas have 4 fields, first two refers to the damage done in adventures and to specific troop, last two represents an generic buff effect coming from Atlas development.

Compila il campo "Class Skill Damage", si riferisce al danno causato da una delle abilità di una classe; controlla l'effettivo valore del danno dal gioco stesso tramite la scheda Abilità prima di entrare.
Poi inserisci i valori di riferimento delle statistiche del tuo personaggio; puoi anche usare valori che non lo sono in tuo possesso per effettuare un confronto tra una o più configurazioni.
Il calcolo del danno può essere effettuato senza la necessità di inserire necessariamente tutti i valori; basti pensare che alcuni dei moltiplicatori come "Damage when combat starts" sono bonus momentanei.
Per calcolare il danno critico di una singola abilità, deve essere compilato il campo "Bonus danno critico", avente valore maggiore di 0.
Se il menu "Danno basato sul tempo" è abilitato, i colpi critici verranno calcolati attraverso una possibilità casuale basata sul campo "Probabilità critica" che apparirà.

Come puoi vedere, il secondo menu al centro contiene altri campi che fanno riferimento ad altri bonus come le statistiche dell'arma, i buff dell'Atlante, il Verdict e così via.

Note:
- Tutte le statistiche sono hardcoded; l'utilizzo involontario di valori maggiori non avrà alcun effetto sul calcolo finale.
- "Elder Multiplier" aumenta il danno fatto da una certa abilità di una classe, ottenuta progredendo all'interno delle Sale degli Elder Gods (massimo 28%).
- Il campo "Generic Class Buff" si riferisce a un'abilità di classe che aumenta i danni, come l'abilità Holy Ground del paladino.
- Il campo "Talento Aelerium-9" si riferisce a un talento di classe che può aumentare il danno.
- Il campo "Bonus simbolo" si riferisce a un simbolo di classe che può aumentare il danno.
- La casella di controllo "Forma divina" simula la forma divina del personaggio, con conseguente aumento dei danni inflitti. Appariranno altri campi collegati allo stato divino.
- Atlas ha 4 campi, i primi due si riferiscono al danno fatto nelle avventure ea una specifica truppa, gli ultimi due rappresentano un generico effetto buff derivante dallo sviluppo di Atlas.

Rules for using effects

Regole per l'uso degli effetti

Unlock the menu, then select an effect type and enter the stack value.
The need to use this section arises from the fact of being able to calculate the damage that can be done with certain Legendary weapons or under certain circumstances.

- "Generic Effect" multiplies the final damage value by 10% for each stack applied (unlimited).
- "Tracer" multiplies the final damage value by 10% for each stack applied (max 16).
- "Viper Mark" multiplies the final damage value for each stack applied by 10% up to 20. After the 20th stack, each new stack adds + 2% to the base multiplier which is 10%, hence the value of the multiplier it will depend on how many stacks you have set (max 100).
- "Dodge That" triples the base damage value of the skill if you previously hit a target with Sureshot, then multiplies the final damage value by 10% for each stack applied (max 16).
- Below you will find more useful things related to support buffs.
- To calculate "Nightfall" buff, you need to set a value for "Companion Damage Bonus" and only then insert the amount of "Static Charges" and the percentage of the stat.

Note:
- "Generic Effect" could be used to calculate the 'Dehydration' skill with Ner'Gal weapon (use max 10 for this).
- Use "Tracer" and "Viper Mark" to calculate the 'Sureshot' skill for the Outlaw class.
- Use "Dodge That" with 0 stack to calculate the 'Dodge That' skill for the Outlaw class. Apply the stacks (max 16) to calculate the damage that the 'Eternal Darkness, Endless Night' weapon can generate.
- "Dodge That" with 0 stack could be used for 'Living Missle' skill calculation via Sigil weapon.

Sbloccare il menu', quindi selezionare un tipo di effetto e inserire il valore dello stack.
L'esigenza di usare questa sezione nasce dal fatto di poter calcolare i danni che si possono effettuare con determinate armi Leggendarie.
- "Generic Effect" moltiplica del 10% il valore finale del danno per ogni stack applicato (senza limiti).
- "Tracer" moltiplica del 10% il valore finale del danno per ogni stack applicato (max 16).
- "Viper Mark" moltiplica del 10% il valore finale del danno per ogni stack applicato fino a 20. Dopo i 20 stack, ogni nuovo stack aggiunge il +2% al moltiplicatore di base che e' del 10%, quindi il valore del moltiplicatore dipendera' da quanti stack avete impostato (max 100).
- "Dodge That" triplica di base il valore del danno della skill se in precedenza hai colpito un bersaglio con Sureshot, quindi moltiplica del 10% il valore finale del danno per ogni stack applicato (max 16).
- Di seguito troverai altre cose utili relative ai buff di supporto.
- Per calcolare il buff "Nightfall", è necessario impostare un valore per "Companion Damage Bonus" e solo successivamente inserire l'ammontare di "Static Charges" e la percentuale della statistica.

Note:
- "Generic Effect" potrebbe essere usato per il calcolo della skill 'Dehydration' tramite arma Ner'Gal (max 10 for this).
- Usare "Tracer" e "Viper Mark" per il calcolo della skill 'Sureshot' per la classe Outlaw.
- Usare "Dodge That" con 0 stack per il calcolo della skill 'Dodge That' per la classe Outlaw. Applicare gli stacks (max 16) per calcolare i danni che puo' generare l'arma 'Eternal Darkness, Endless Night'.
- "Dodge That" con 0 stack potrebbe essere usato per il calcolo della skill 'Living Missle' tramite arma Sigil.
