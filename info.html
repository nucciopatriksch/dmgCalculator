Rules for using the calculator

Regole per l'utilizzo del calcolatore

Enter the value of the damage caused by one of the skills of a class, for example Revenant skill Slaughter; check the actual damage value from the game itself via the Abilities tab before entering.
Then insert the reference values of your character's stats; you can also use values that are not in your possession to make a comparison between one or more configurations.
The calculation of the damage can be done without the need to necessarily enter all the values; just think that some of the multipliers such as "Damage when combat starts" are momentary bonuses.
To calculate critical damage, the "Critical damage bonus" value must be greater than 0.
If "Time based Damage" menu is enabled, Critical hits will be calculated basing on "Critical Chance" stat.

Note:
- "Base Damage" refers to your character's base multiplier (max 56%).
- "Elder Multiplier" increases the damage done by a certain skill of a class, obtained by progressing within the Halls of Elder Gods (max 28%).
- "Damage when combat starts" increases your damage at the start of a battle and for the first 15 seconds (max 59%).
- "Artifact Damage bonus" increases all damage done, obtained through a Legendary Artifact (max 35%).
- "Class Seal Bonus" increases all damage done by a class, obtained through the Cathedral (max 10%).
- "Adventure Seal Bonus" increases all damage done in a certain type of adventure, obtained through the Cathedral (max 5%).

Inserire il valore del danno causato da una delle skill di una classe, ad esempio Revenant skill Slaughter; controllare l'effettivo valore del danno dal gioco stesso tramite la scheda Abilities prima dell'inserimento.
Inserire quindi i valori di riferimento alle statistiche del vostro personaggio; potete usare anche valori non in vostro possesso per effettuare un confronto tra una o più configurazioni.
Il calcolo del danno può essere effettuato senza la necessità di inserire necessariamente tutti i valori; basti pensare che alcuni dei moltiplicatori come ad esempio "Damage when combat starts" sono bonus momentanei.
Per calcolare i danni critici il valore di "Critical damage bonus" deve essere maggiore di 0.
Abilitando il menu "Time based Damage", i Colpi critici saranno calcolati in base alla stat "Critical Chance".

Note:
- "Base Damage" si riferisce al moltiplicatore di base del vostro personaggio (max 56%).
- "Elder Multiplier" aumenta i danni effettuati da una certa skill di una classe, si ottiene facendo progressi all'interno della Halls of Elder Gods (max 28%).
- "Damage when combat starts" aumenta i vostri danni all'inizio di una battaglia e per i primi 15 secondi (max 59%).
- "Artifact Damage bonus" aumenta tutti i danni effettuati, si ottiene tramite un Artefatto Leggendario (max 35%).
- "Class Seal Bonus" aumenta tutti i danni effettuati da una classe, si ottiene tramite la Cattedrale (max 10%).
- "Adventure Seal Bonus" aumenta tutti i danni effettuati in un determinato tipo di avventura, si ottiene tramite la Cattedrale (max 5%).

Rules for using effects

Regole per l'uso degli effetti

Unlock the menu, then select an effect type and enter the stack value.
The need to use this section arises from the fact of being able to calculate the damage that can be done with certain Legendary weapons or under certain circumstances.

- "Generic Effect" multiplies the final damage value by 10% for each stack applied (unlimited).
- "Tracer" multiplies the final damage value by 10% for each stack applied (max 16).
- "Viper Mark" multiplies the final damage value for each stack applied by 10% up to 20. After the 20th stack, each new stack adds + 2% to the base multiplier which is 10%, hence the value of the multiplier it will depend on how many stacks you have set (max 100).
- "Dodge That" triples the base damage value of the skill if you previously hit a target with Sureshot, then multiplies the final damage value by 10% for each stack applied (max 16).

Note:
- "Generic Effect" could be used to calculate the 'Dehydration' skill with Ner'Gal weapon (use max 10 for this).
- Use "Tracer" and "Viper Mark" to calculate the 'Sureshot' skill for the Outlaw class.
- Use "Dodge That" with 0 stack to calculate the 'Dodge That' skill for the Outlaw class. Apply the stacks (max 16) to calculate the damage that the 'Eternal Darkness, Endless Night' weapon can generate.
- "Dodge That" with 0 stack could be used for 'Living Missle' skill calculation via Sigil weapon.

Sbloccare il menù, quindi selezionare un tipo di effetto e inserire il valore dello stack.
L'esigenza di usare questa sezione nasce dal fatto di poter calcolare i danni che si possono effettuare con determinate armi Leggendarie.
- "Generic Effect" moltiplica del 10% il valore finale del danno per ogni stack applicato (senza limiti).
- "Tracer" moltiplica del 10% il valore finale del danno per ogni stack applicato (max 16).
- "Viper Mark" moltiplica del 10% il valore finale del danno per ogni stack applicato fino a 20. Dopo i 20 stack, ogni nuovo stack aggiunge il +2% al moltiplicatore di base che è del 10%, quindi il valore del moltiplicatore dipenderà  da quanti stack avete impostato (max 100).
- "Dodge That" triplica di base il valore del danno della skill se in precedenza hai colpito un bersaglio con Sureshot, quindi moltiplica del 10% il valore finale del danno per ogni stack applicato (max 16).

Note:
- "Generic Effect" potrebbe essere usato per il calcolo della skill 'Dehydration' tramite arma Ner'Gal (max 10 for this).
- Usare "Tracer" e "Viper Mark" per il calcolo della skill 'Sureshot' per la classe Outlaw.
- Usare "Dodge That" con 0 stack per il calcolo della skill 'Dodge That' per la classe Outlaw. Applicare gli stacks (max 16) per calcolare i danni che può generare l'arma 'Eternal Darkness, Endless Night'.
- "Dodge That" con 0 stack potrebbe essere usato per il calcolo della skill 'Living Missle' tramite arma Sigil.
